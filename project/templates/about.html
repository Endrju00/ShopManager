Products Component

Contains and allows you to define new information about wholesalers, manufacturers, categories, products and supplies.


Usage

It allows you to view, add, edit and delete information about the above-mentioned topics.

Orders Component

Contains and allows you to define new information about orders, items in orders, addresses and payments.


Usage

It allows you to view, add, edit and delete information about the above-mentioned topics. To view and edit all addresses go to Report component.

Employees Component

Contains and allows you to define new information about employees and their positions.



Usage

It allows you to view, add, edit and delete information about the above-mentioned topics.

Clients Component

Contains and allows you to define new information about clients.


Usage

It allows you to view, add, edit and delete information about the above-mentioned topics.

Report Component

Contains and allows you to view compressed information about everything in your store.




Usage

It allows you to view tables containing compressed information (without descriptions and comments) about your store. You can edit Addresses by clicking on them.

Use Guide

To add the new object go to their section in their component and click the button
which is located next to the searchbar at the top of the list.


To edit the object go to their section in their component and click on it. Each detail view is equipped with the edit button
which is located at the top right corner of the detail page.


To delete the object go to their section in their component and click on it. Each detail view is equipped with the delete button
which is located at the top right corner of the detail page.


To add an item to order that already exists go to their detail view. Each order detail view is equipped with the button
which is located at the top right corner of the detail page.

General info

The application is used to manage the store. This system will enable the storage of information about products, their stock in the warehouse, orders from customers, information about customers, employees and wholesalers from which the store orders the goods. The user who uses the application will be able to define, view and edit this data. For example, it can enter information about new orders, products, and also change, for example, employee wages.
